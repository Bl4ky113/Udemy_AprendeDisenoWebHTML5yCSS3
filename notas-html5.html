Etiquetas y Atributos En Html 5

En html se usa el sistemas de etiquetas, estas funcionan para que el navegador en cuestión
intérprete las acciones que se le programan o se le ordenan de una forma grafica en la página web.
Para escribir una etiqueta toca tener en cuenta la siguiente estructura:

< "InicioEiqueta" > Contenido Etiqueta < /"CierreEtiqueta" >

A estas Etiquetas se les puede agregar atributos, para información, reglas o ordenes extra para el navegador, estas
van dentro de la etiqueta de inicio o < "InicioEtiqueta" >. Siguiendo la forma de la estructura, quedaria asi:

< "InicioEtiqueta" atributo="valorAtributo" > contendio Etiqueta < /"CierreEtiqueta">

Algunos atributos son Bolean, osea solo tienen como valor a true o false
Eso nos permite que con solo escribir el atributo, sin ningun valor, este funcione

< "InicioEtiqueta" atributoBoleano > Contenido Etiqueta < /"CierreEtiqueta" >

Links o Enlaces de html

Con la etiqueta "a" se puede crear links o enlaces a otras direcciones web, pueden ser internas o externas a tu sitio web.
Esta etiqueta tiene el atributo "href", donde se indica la URL de la pagina web a donde va a direccionar este link.
Tambien se puede usar un atributo llamado "target", donde se indica como se va a desplegar la página web que direcciona el link.
Los valores de este atributo son:

  • _blank

    Abre el link de la página web, en otra pestaña del navegador

  • _self

    Abre el link de la página web, en la misma pestaña en donde se abrió la página web

Listas en Html5

En html existen 3 tipos de listas, estas pueden ser usadas en diferentes circunstancias y con diferentes objetivos, estas son:

Lista Ordenada

Para esta lista se usa la etiqueta "ol" para crear la lista y cada objeto de esta se usa la etiqueta "li"
esta tiene 3 atributos muy interesantes
El primero Start, define donde va a iniciar el conteó de la lista.
El segundo type, define en que tipo se va a contar & enumerar la lista, esta tiene solo unos 5 valores posibles, que son:

  1. "A" o "a"

    Enumera la lista en forma alfabetica. (a, b, c, d... O A, B, C, D...).

  2. "I" o "i"

    Enumera la lista en Números romanos. (I, II, III.. o i, ii, iii...).

  3. 1

    Enumera la lista en números normales. (1, 2, 3, 4, 5...)

Y el tercer reversed, no tiene valor, pero hacer que el conteó de la lista inicie alrevez
ejemplo:

  2. Cebollas
  3. Pepinos
  4. Lechuga
  5. Limones

Meta Datos

Los metadatos son informacion que el navegador va a leer y utilizar al momento de mostrar el documento
Estos metadatos pueden tener informacion de que tipo de caracteres que se va a usar (UTF-8 es el standard mundial)
Pueden tener informacion de los creadores y del proyecto & sitio/pagina web.
Y puede tener informacion en como se va a mostrar dependiendo de en que dispositivo se vaya a abrir el documento (desk, cell, tablet, etc.)

Estos metadatos se les puede dar a el navegador en la parte del "Head" con la etiqueta "meta"
Esta etiqueta tiene unos atributos que definen el tipo y la informacion del dato que se le va a entregar a el navegador
Ejemplos

  • Definir el tipo de caracter:
    name="UTF-8"

  • Definir Informacion del autor:
    name="author" content="Joe Mama"

  • Definir la descripcion del sitio/pagina web:
    name="description" content="Descripcion sitio web"
    Esta descripcion ayuda mucho a los buscadores como Google a encontrar y recomendar tu pagina & sitio web

  • Definir las "Keywords" del sitio/pagina web
    name="keywords" content="etiquetas o keywords"
    Estas keywords ayuda mucho a los buscadores como Google a encontrar y recomendar tu pagina & sitio web

  • Definir el "viewport" o como va a ver el usuario el documento
    name="viewport" content="width=(ancho de la pantalla del usuario) initial-scale=(La escala del zoom inicial del documento)"
    Generalmente, se usa en este content width="device-width" initial-scale=1.0, para que obtenga el ancho de la pantalla del usuario y se ajusta el documento a esta.

Main en la Estructura

El main va ser donde esta el contenido de la pagina web, tipo algun articulo, aviso, elementos, etc.
Puede ser considerado como el elemento principal del body de una pagina web, este no puede ser usado más de una vez en todo el documentos, o se consideraria una mala practica
Generalmente en el Main no va solo, sino que este se divide en diferentes secciones, articulos, avisos, o subdivisiones de la pagina web, etc.

Nav en la Estructura

El Nav es una parte importante de la Estructura
Este contiene diferentes links del Sitio web, donde nos redirigen a diferentes partes del Sitio
Esto nos permite que el usuario tenga un menu de navegación del sitio web, y pueda tener una interfaz mucho más sencilla

Generalmente el nav se incluye en elementos importantes, cómo el header, main (a un lado) y Footer.

En el Header, puede ir al lado del logo y titulo & puede ir debajo de este, puede tener los articulos & categorias principales del sitio web
En el Main, puede ir un rectangulo vertical a un lado de la pantalla (Right || Left), puede tener todas las categorias y algunos articulos del sitio web
En el Footer puede ir en el top o en el inicio de este, puede tener todas las categorias y articulos del sitio web
pero generalmente solo mantiene información de derechos de autor, contactos y demás temas legales.

Secciones en la Estructura

El Section es la etiqueta que se usa cuando se necesita dividir un contenido de una parte del documento
Formando diferentes secciones, como por ejemplo un super mercado, cada pasillo es una seccion
Dividiendo todo el contenido en diferentes secciones que por si solas pueden funcionar sin depender del resto
En los sections hay tambien pequeñas divisiones de contenido, estas se les llama Articulos.

Articulos en la Estructura

Los articulos, como anteriormente dicho, son subdivisiones del elemento section
Estos igualmente que el section, pueden existir sin apoyo, informacion o existencia de otros elementos
Retomando el ejemplo del super mercado, el article dentro de los pasillos, pasa a ser el articulo o el objeto que puedes comprar

Aside en la Estructura

El Aside es una etiqueta que se usa para separar contenido & informacion del contenido principal
Su objetivo principal es agregar informacion que puede aportar a el contenido principal, pero que no es totalmente necesario
ejemplo de los contenidos de un aside:

  • Informacion Relacionada
  • Contenido Relacionado

  • Entre otros...

Es muy Importante no confundir el Nav con el aside, son totalmente diferentes !!!
Pueden existir varios asides en el documento, dentro de Sections, Articles o hasta el Main, pero nunca se les debe usar como un Nav

Esto que hace? Hace que la etiqueta Strong tenga un signidicado al usarse, además de añadir negrita a el texto en general.

Ya si no es texto importante, o por alguna razon no puedes usar CSS para cambiar el style del texto, usa la etiqueta b
Cabe aclarar que usando b, Strong o El Style de CSS, vas a obtener el mismo resultado si no modificas el style del objeto.

Video en Html5

La etiqueta video, funciona para poder anexar un video a nuestra pagina web
si algun tipo de video no es soportado por un navegador o si ese tipo de archivo no carga o esta caido
se va a mostrar el texto o el contenido escrito entre la etiqueta

Esta etiqueta tiene muchos atributos para su buen funcionamiento, estos atributos son:

  • src="URL"
    Se usa para darle la URL del video a el elemento
    aunque se puede usar la etiqueta source dentro del elemento para tener multiples fuentes
  • height/width="tamaño en pixeles"
    Se usa para determinar el tamaño del reproductor de video
    Asegurate que sea un rectangulo y que el video si se pueda ver bien
  • controls="Controles"
    Se usa para definir si se muestran o se ocultan los controles del reproductor de video
    puedes elegir cuales si y cuales no se pueden mostrar, esta es la lista de controles
    • play
    • Pause
    • Seeking (Barra de progreso)
    • Volume
    • Fullscreen toggle
    • Captions/Subtitles
    • Track
    Si quieres poner todos, puedes poner el atributo sencillo, sin ningun valor.
  • preload="auto|metadata|none"
    Define si se quiere que el video carge con la pagina, carge solo la metadata o que solo cargue cuando el usuario inicia el video
    Esto puede afectar o ayudar a el desempeño de las paginas web.
  • loop="Loop"
    Define si el video se va a repetir cada vez que termina
    Se puede usar sin ningun valor, sencillo
  • autoplay="AutoPlay"
    Define si el video va a iniciar una vez que este haya cargado, no importa el atributo preload
    Se puede usar sin ningun valor
  • muted="Muted"
    Define si el video va a estar muteado
    Se puede usar sin ningun valor
  • poster="URL-Imagen" Usa una imagen como poster para cuando el video este cargando
    incompatible con autoplay

Con JS podemos modificar la reproduccion del video, dandole un mejor efecto y uso a el usuario

Se puede usar videos de youtube usando la etiqueta iframe

¿Que es Canvas?

Es una API que funciona para dibujar graficos, composiciones de fotos, animaciones o aplicaciones, generalmente usando JavaScript. De hecho canvas viene a sustituir tecnologias como Flash Media y otras.

Para tener un mejor uso de canvas, tenemos que saber que canvas usa un sistema de coordenadas como el plano cartesiano. Este inicia en la esquina superior de la izquierda. Dando como marca el punto (0,0), 0 en el eje X y 0 en el eje Y.

Usar Canvas

Para usar Canvas debemos usar la etiquta canvas, este no se va a ver nada, ya que no tiene ningun contenido. Por defecto siempre va a tener un ancho y alto, pero siempre le debemos asignar estos datos, ya que algunos navegadores cambian la altura y anchura del canvas por defecto.

Vamos a poder ser un artista como un lienzo, pero no usando un pincel si no JavaScript, este lo vamos a agregar a el documento de html. Ya sea escribiendolo en la etiqueta o usando un archivo externo.

vamos a reconocer el canvas en JavaScript usando el artibuto id="", creando una variable y obteniendo el ID del canvas.

let canvas = document.getElementById("idCanvas");
//Se identifica y obtiene el Canvas por su Id.

Listo ahora necesitamos un "lapiz" virtual, para poder dibujar en el canvas. Para eso lo creamos dandole un contexto de dibujo a el canvas, generalmente 2d. Casí siempre se le va a llamar a el "lapiz" ctx.

let ctx = canvas.getContext("2d")
//Se le da un tipo de dibujo en 2d a el Canvas.

Para darle un uso sencillo, vamos a usar el sistema de coordenadas para hacer un rectangulo, primero llamamos el pincel, usamos la funcion para hacer rectangulos y agregamos nuestras coordenadas de la siguiente forma: (X inicial, Y inicial, Tamaño de X, Tamaño de Y). En las coordenadas iniciales son donde van a iniciar el rectangulo y en las finales, que tanto crecera el rectangulo.

ctx.fillRect(0, 0, 50, 50);

Diferentes Figuras en Canvas

En canvas podemos usar algunas funciones ya hechas para poder tener diferentes figuras. Aún que la mayoria de estas son rectangulos o cuadrados.

Rectangulo Relleno

Se usa para crear un rectangulo relleno, en sus parametros vamos a poner la coordenada X donde va a iniciar el rectangulo, la coordenada Y donde va a iniciar el rectangulo. Y que tan grande va a ser el rectangulo en el eje X y en el eje Y. (Xinicial, Yinicial, Xtamaño, Ytamaño). Ejemplo:

// Dibujar el Rectangulo
rectanguloRelleno.fillRect(10, 10, 90, 90);

Rectangulo Vacio

Es lo mismo que el rectangulo anterior, pero este eliminara o borrara espacios con el tamaño de un rectangulo. Muy bueno usarlo dentro de rectangulos rellenos, para dar diferentes formas en el canvas.

//Dibujar un Rectangulo
ctx.fillRect(10, 10, 80, 80);
//Eliminar parte el Rectangulo
ctx.clearRect(20, 20, 60, 60);

Rectangulo Con Bordes

Con esta se puede crear rectangulos que tengan sus propios bordes, evitando que tengamos que hacer un rectangulo y depues quitarle parte de este. Se puede cambiar el ancho, color y otros del borde. Pero esto lo vamos a ver más tarde. Mientras tanto, tiene los mismos parametros que las funciones anteriores, por lo tanto no debería ser tan dificil de manejar.

//Rectangulo con bordes
ctx.strokeRect(20, 20, 60, 60);

Crear o Hacer otras Figuras.

Para hacer nuestras propias figuras debemos ussar varias funciones para esto. Primero vamos a crear el lapiz, ya antes mencionado, con:

let canvas = document.getElementById("id-canvas"); let ctx = getContext("2d");

Como lo hacemos en la vida real, vamos a iniciar un trazo para hacer una recta o segmento.

ctx.beginPath();

Vamos a mover el lapiz a las coordenadas que necesitemos.

ctx.moveTo(Xi, Yi);

Y vamos a trazar el segmento, dando las coordenadas a donde este segmento vaya a ir.

ctx.lineTo(Xf, Yf);

Despues de haber hecho todos los segmentos necesarios, vamos a cerrar el trazo para evitar problemas con futuros trazos.

ctx.closePath();

Pero ojo, que en el canvas se va a hacer todo el trazo, pero no se mostrara. Para esto podemos rellenarlo & darle un borde.

// Rellernarlo
ctx.fill();
// Darle un Borde
ctx.stoke();

Styles en Canvas

Para agregarle un style a nuestro canvas, podemos usar diferentes cosas desde colores, anchos de lineas, entre otros. Estos Styles deben ser declarados antes de ser usados en las diferentes acciones que hay, estas pueden ser desde los bordes o rellenos.

Color de Relleno

Este se usa para las figuras rellenadas, ya sea con "fillRect()" o "fill()". En esta función se puede poner cualquier color de CSS o un Canvas Graident, Canvas Pattern.

// Rellenar de color
ctx.fillStyle = <"Color">

Color del Borde

Este se usa para las figuras que tienen borde, ya sea con "strokeRect()" o "stroke()". En esta función se puede usar cualquier color de CSS o un Cavas Graident, Canvas Pattern. Talvez no se note un poco el borde, pero para poder cambiar el style del Borde también existen diferentes Styles.

// Colear el Borde
ctx.strokeStyle = <"Color">

Styles de los Bordes & Lineas

Se puede dar diferentes cambios a el Style a las Lineas o bordes de las figuras, estas son:

  • ctx.lineWidth = <"Número">;
    Define el ancho de la linea o del borde. Medida en px o pixeles.
  • ctx.lineCap = butt || round || square;
    Define el style del final de la linea, ya sea redondeada, cuadrada o con un cuadrado al final.
  • ctx.lineJoin = bevel || round || miter;
    Define el Style de los vertices entre dos lineas, ya sea plano, redondeado o cuadrado.
  • ctx.miterLimit = <"Número">
    Define el limite de radio que tienen los vertices entre dos lineas, generalmente 10 es el predeterminado.

Gradiantes en Canvas

Estos son colores que se le pueden aplicar a las figuras del Canvas. Para realizarlos, debemos crear una variable para la gradiante, existen diferentes tipos para crear una gradiante, esta es en una sola linea. Como si fuera una linea, nos va a pedir los siguientes parametros:

  • X inicial
  • Y inicial
  • X Final
  • Y Final

Generalmente queremos que nuestra gradiante ocupe todo el canvas, para eso podemos usar unos valores de la variable de canvas, para usarlos acá: canvas.height y canvas.width, nos va a dar respectivamente el alto y ancho del objecto. Ojo, este truco solo funciona si tienen un objecto con el elemento del canvas, no si lo tienes ya hecho con context, tampoco se puede hacer si no tienes una anchura y altura ya puesta, ya que se toma el valor de los atributos, más no del canvas en general

let grandiante = ctx.createLinearGradient(xi, yi, xf, yf);

Despues vamos a formar nuestra gradiante, esta debe tenre 2 colores, uno al iniciar y otro al terminar, la funcion que vamos a usar va a pedir 2 parametros sencillos, el número del color (0 = inicio; 1 = final) y el color de CSS para usarlo.

gradiante.addColorStop(0, black);
gradiante.addColorStop(1, white);

Listo! ahora podemos usar nuestra gradiante como si fuera un color. Yo lo voy a usar para colorear un cuadrado que ocupe todo el canvas

ctx.fillStyle = grandiante;
ctx.fillRect(0, 0, canvas.width, canvas.height);

Geolocalización por Coórdenadas

Vamos a usar una API, para poder obtener nuestra Geolocalización en Coórdenadas. Para esto simplemente vamos a tener que usar un poco de JavaScript, un lugar donde poner núestras coórdenadas y un botón para que busque y tome nuestras coórdenadas.

Para hacerla, vamos a usar un button con una función que se active cuando le demos "click".

< button onlick="getLocation();"> "texto del button" < /button >

Esto va a ser un iniciador del script, pero vamos a necesitar un lugar donde mostrar las coordenadas. Creando otra etiqueta con un atributo ID.

< p id="location-output"> Texto < /p >

Aquí el texto dentro de la etiqueta puede no importar mucho, ya que generalmente vamos a cambiarlo y borrarlo por otros contenidos.

Teniendo todo, ahora vamos a hacer el script. Vamos a inciar por obtener la salida de los datos, que es la etiqueta p con el atributo ID. Lo que le vamos a decir a el navegador es:

  1. Nombra esta variable "output"
  2. Busca en el documento un elemento con el atributo ID "location-output"
  3. Este elemento asignalo a la variable "output"
let output = document.getElementById("location-output");

Ahora vamos a hacer la funcion con el API de Geolocalización de la siguiente forma.

function getLocation () {

}

Dento de los "brackets", vamos a primero verificar si el navegador soporta el API de geolocalizacion de la siguiente forma:

if (navigator.geolocation) {
/* Si sí lo soporta */
} else {
/* Si no lo soporta */
}

Ahora si lo soporta vamos a usar la API para que use una funcion y nos devuelva el valor en otra función

navigator.geolocation.getCurrentPosition("funcion");

Si no lo soporta vamos a escribir en el HTML interno de la salida, que no lo soporta.

output.innerHTML = "No soporta la API de geolocalización";

En la funcion o una inline-function, que va a tener el valor de nuestras coordenadas, vamos a mostrarla en el output, ingresando a el objeto de coordenadas y mostrando su latitud y longitud.

function "funcion" (posicion) {
output.innerHTML = "Latitud: " + posicion.coords.latitude +
" Longitud: " + posicion.coords.longitud;
}
